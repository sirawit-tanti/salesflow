# SalesFlow - Quotation, Invoice & Payment Management System

SalesFlow is a full-stack portfolio project for managing customers, quotations, invoices, payments, receipts, reports, and audit logs.

## Tech Stack

### Backend

- Laravel 12 REST API
- MySQL
- Laravel Sanctum
- Role-based access control

### Frontend

- React
- TypeScript
- Vite
- Tailwind CSS

## Core Workflow

Customer
→ Create Quotation
→ Send Quotation
→ Customer Accepts
→ Convert to Invoice
→ Record Payment
→ Generate Receipt
